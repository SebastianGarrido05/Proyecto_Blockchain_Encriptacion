Como usar
Desde el cliente darán dos opciones
1.- Cifrar
2.- Descifrar

En caso de querer cifrar se debe poner:
" python crypto_tool.py cifrar nombre_archivo.txt "

En caso de querer descifrar se pone:
" python crypto_tool.py descifrar nombre_archivo.bin "
